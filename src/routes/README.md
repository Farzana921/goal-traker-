# GoalForge 🎯

GoalForge is a personal goal-tracking web application that helps users organize goals, track progress, build streaks, and stay motivated.

## Features

* Create and manage goals
* Track progress and completion
* XP and level system
* Goal categories
* Progress analytics
* English and Persian (RTL) support
* Local storage persistence

## Built With

* React
* TypeScript
* TanStack Router
* Tailwind CSS
* shadcn/ui
* Recharts

## About

I built this project to practice modern React development and create a clean productivity tool for tracking personal goals and habits.

More features and improvements will be added in future updates.
